## ğŸ¨ Customization

### Update Personal Information
Edit `js/components/config.js` to update:
- Name, email, and social media links
- EmailJS configuration
- Profile image paths

### Add/Modify Projects
Edit `js/components/projects.js` to:
- Add new projects
- Update project descriptions, tags, and links
- Configure project images for carousels

### Add Images
See `IMAGE_GUIDE.md` for detailed instructions on:
- Adding profile photos
- Adding project screenshots
- Image optimization tips
- Troubleshooting image issues

- `tailwind.config.js`: Tailwind customization (colors, fonts, etc.)
- Font Awesome icons can be changed in `index.html`
